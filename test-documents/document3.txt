n8n - это платформа автоматизации с открытым исходным кодом для создания рабочих процессов.

n8n позволяет пользователям создавать автоматизированные процессы, соединяя различные сервисы и API через графический интерфейс.

Ключевые особенности n8n:
- Визуальный редактор рабочих процессов
- Более 400 готовых интеграций
- Поддержка пользовательских узлов (custom nodes)
- Самостоятельное размещение (self-hosted)
- Поддержка JavaScript для сложной логики
- Обработка ошибок и повторные попытки
- Планировщик задач и триггеры

n8n широко используется для автоматизации бизнес-процессов, интеграции данных, мониторинга систем и создания ETL-пайплайнов. Платформа особенно популярна среди разработчиков благодаря своей гибкости и возможности расширения.
